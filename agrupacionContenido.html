Etiqueta div

Define una división o una sección en un documento html, se puede poner cualquier tipo de contenido

Etiqueta pre

Define texto preformateado. El texto se muestra en una fuente de ancho fijo y conserva tanto los espacios como los saltos de linea. Ejemplo: 

                Text in a pre element
                is displayed in a fixed-width
                font, and it preserves
                both spaces and
                line breaks

Etiqueta blockquote

Define una cita larga. Ejemplo:

He aquí una cita del sitio web de WWF:
Desde hace 50 años, WWF protege el futuro de la naturaleza. WWF, la principal organización conservacionista del mundo, trabaja en 100 países y cuenta con el apoyo de 1,2 millones de miembros en Estados Unidos y cerca de 5 millones en todo el mundo.

Etiqueta dl, dt & dd

dl: Define una lista de descripción, dt: Define un termino en una lista de descripcion, dd: Describe un termino en una lista de descripción.

Ejemplo:

Coffee
Black hot drink
Milk
White cold drink

Etiqueta ol & li

Define una lista ordenada, una lista ordenada puede ser numérica o alfabética. Se utiliza li para definir cada elemento de la lista. Ejemplo:

  2. Verano
  3. Otoño
  4. Invierno

Etiqueta ul

Define una lista desordenada con viñetas, usamos la etiqueta li para crear listas desordenadas

  • Primavera
  • Verano
  • Otoño
  • Invierno

Etiqueta figure

Etiqueta especifica contenido independiente, como ilustraciones, diagramas, fotos, listas de códigos, etc.

Etiqueta figcaption

Define un titulo para un elemento figure, se puede colocar como el primer o ultimo hijo del elemento.
