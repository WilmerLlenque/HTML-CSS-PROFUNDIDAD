Etiqueta span

Es un contenedor en línea que se utiliza para marcar una parte de un texto o una parte de un documento. Ejemplos:

° We can use span tags to target text with CSS styling, like with this red text and this green text.

° You can use span for any CSS rule. Here I'm using the background-color property to highlight text.

Etiqueta a

Define un hipervínculo, que se utiliza para enlazar de una página a otra. Ejemplos:

Platzi Redirecciona a mi Footer Contactame

Etiqueta dfn

Es un elemento de definición. Ejemplo:

HTML es el lenguaje de marcado estándar para la creación de paginas web

Etiqueta abbr

Define una abreviatura o un acrónimo. Ejemplo:

CSS es un lenguaje que describe el estilo de un documento html.

La OMS fue fundada en 1948

Etiqueta q

Define una cita corta, los navegadores normalmente insertar comillas alrededor de la cita. Ejemplo:

El objetivo de WWF es: Construir un futuro donde las personas vivan en armonía con la naturaleza. Esperamos que tengan éxito.

Etiqueta cite

Define el título de una obra creativa (por ejemplo, un libro, un poema, una canción, una película, una pintura, una escultura, etc.). Ejemplo:

El principito por Antoine de Saint-Exupéry. Publicado en 1943.

Etiqueta em

Se utiliza para definir texto enfatizado, un lector de pantalla pronunciará las palabras con énfasis. Ejemplo:

You have to hurry up!

We cannot live like this.

Etiqueta time

Define una hora especifica (o fecha y hora). Ejemplo:

Abierto de 10:00 a 21:00 todos los dias.

Tengo una cita en el dia de San Valentin

Etiqueta var

Define una variable en una expresión matemática o de programación. Ejemplo:

El area de un triangulo es 1/2*b*h, donde b es la base y h es la altura vertical.

Etiqueta samp

Se utiliza para definir la salida de muestra de un programa de computadora. El contenido del interior se muestra en la fuente monoespaciada predeterminada del navegador.Ejemplo:

Archivo no encontrado, presione F1 para continuar

Etiqueta b

Especifica texto en negrita sin ninguna importancia adicional. Ejemplo:

Este texto es normal y este texto en negrita.

Etiqueta sub & sup

Etiqueta sub: El texto del subíndice aparece medio carácter por debajo de la línea normal y, a veces, se representa en una fuente más pequeña. El texto de subíndice se puede utilizar para fórmulas químicas, como H 2 O.

Etiqueta sup: El texto en superíndice aparece medio carácter por encima de la línea normal y, a veces, se representa en una fuente más pequeña. El texto en superíndice se puede utilizar para notas al pie, como WWW [1] .

Ejemplos:

Este texto contiene texto subindice

Este texto contiene texto superindice

Etiqueta small

Define texto más pequeño (como derechos de autor y otros comentarios secundarios). Ejemplo

Este es un texto normal, este es un texto pequeño

Etiqueta strong

Se utiliza para definir texto con gran importancia. El contenido interior normalmente se muestra en negrita. Ejemplo

Este es un texto importante

Etiqueta mark

define el texto que debe marcarse o resaltarse. Ejemplo

No olvides comprar leche para el desayuno

Etiqueta ins & del

Etiqueta ins: Define un texto que se ha insertado en un documento. Los navegadores generalmente subrayarán el texto insertado.

Etiqueta del: Define el texto que se ha eliminado de un documento. Los navegadores generalmente marcarán una línea a través del texto eliminado.

Ejemplos:

Mi color favorito es Azul Rojo.

Etiqueta s

Especifica texto que ya no es correcto, preciso o relevante. El texto se mostrará con una línea que lo atraviesa.

Solo quedan 50 entradas

Etiqueta code

Define un fragmento de código dentro de un bloque de texto.

Escriba npm install en su terminal para instalar las dependencias de un proyecto.
